# Documentation Assets

This directory contains images and assets for the README documentation.

## Screenshots Needed

To complete the README documentation, please add the following screenshots:

1. **dashboard-preview.png** - Screenshot of the main dashboard showing the API key management interface
2. **logs-preview.png** - Screenshot of the live logs view

## How to Add Screenshots

1. Take screenshots of your running application
2. Save them in this directory with the names mentioned above
3. The README will automatically display them

## Image Guidelines

- Format: PNG (preferred) or JPG
- Resolution: 1280x720 or higher recommended
- File size: Keep under 500KB for faster loading
